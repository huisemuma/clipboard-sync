剪贴板同步 — 手机语音输入，电脑直接粘贴
============================================

用途：
  手机上用语音输入打字 → 点发送 → 电脑 Cmd+V 直接粘贴
  （配合豆包输入法等语音输入法使用，效果最佳）

使用方法：
  1. 双击「启动剪贴板同步.command」
  2. 终端会显示一个地址，类似 http://192.168.x.x:9898
  3. 手机浏览器打开这个地址（手机和电脑需在同一 WiFi）
  4. 在网页里语音输入或打字，点发送
  5. 电脑上 Cmd+V 粘贴即可

前置要求：
  - Mac（Apple Silicon / M芯片）
  - Node.js（首次运行如果没装会提示安装方法）
  - 手机和电脑在同一 WiFi 下

首次运行会自动安装依赖，之后秒开。
用完了直接关终端窗口即可。
